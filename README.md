# Kişisel Finans Yönetimi

Kendi kullanımım için geliştirdiğim basit bir masaüstü finans takip uygulaması.

## Kurulum

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

## Özellikler

- Çoklu para birimi (TRY, USD, EUR)
- Hesap yönetimi
- Gelir/gider takibi
- Planlanan işlemler

## Teknoloji

- Python 3
- PyQt6
- SQLite
